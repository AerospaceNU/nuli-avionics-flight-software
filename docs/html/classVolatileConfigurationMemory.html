Detailed Description

template<unsigned N>
class VolatileConfigurationMemory< N >

For board without non-volatile memory for configuration.

Allows the configuration API to be used for boards without non-volatile memory, however it will reset every time the code restarts. This is needed because the configuration API is guaranteed to be available.

NSize of the memory
