Accelerometer * ICM20602Sensor::getAccelerometer ( )

Gets the Accelerometer injector class.

Injector classes are used to pass data to the HardwareAbstraction. The ICM20602 is basically a 3 in one sensor, so we make a "dummy" class for each of the 3 sensors, and pass data through it.

Reads data from the sensor.
