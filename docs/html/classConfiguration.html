void Configuration::setup ( HardwareAbstraction * hardware,
uint8_t id )

Reads in the Configuration from FRAM.

Checks that the configuration passes a CRC check, version number check, and a CRC check on all the IDs present (to track if the set of configuration varables has changed). If any errors are present, it restores the default values.

Parameters
hardwareHardwareAbstraction to retrieve the FRAM and DebugStream from
idIndex of the FRAM in the HardwareAbstraction FRAM array
