Detailed Description

Underlying interface to all sensors.

Provide virtual methods for common sensor function. Currently this is not a used feature, but is implemented for future code that may require a BaseSensor class

Read data from the sensor.

Read in one reading from the sensor, and convert the data to usefully units/numbers. Currently is allowed to block the loop to wait for data from the sensor for a few ms.

Reimplemented in ArduinoPyro, ArduinoVoltageSensor, ICM20602Sensor, ICM20948Sensor, MS5607Sensor, MS8607Sensor, Pyro, UART_GPS, and VoltageSensor.
