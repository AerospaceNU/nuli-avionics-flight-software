Detailed Description

Utility class to debounce boolean signals.

This class provides a mechanism to debounce signals, typically used to prevent spurious transitions in digital inputs due to noise or mechanical switch bounce. It ensures that a condition must remain true for a specified amount of time before it is considered valid.

bool Debounce::check ( const bool condition,

inline

Check if a condition has been stable for the debounce time.

When called with a boolean condition and the current timestamp, this function returns true only if the condition has been continuously true for at least the debounce time specified during construction or via setDebounceTime(). If the condition is false, the internal reference time is reset.

Parameters
conditionThe boolean condition to debounce.
timestampThe current timestamp as a Timestamp_s object, where runtime_ms represents the elapsed time in milliseconds.
Returns
true if the condition has been true for the debounce duration, false otherwise.

Reset the debounce state.

This function clears the internal state, including the reference time and initialization flag. After calling reset(), the next call to check() will treat the condition as if it were the first time being checked.
