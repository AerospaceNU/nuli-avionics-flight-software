Detailed Description

template<typename T>
class ConfigurationData< T >

Wrapper object for users to interact with configuration variables.

Tracks updates, allowing for automatic writes to FRAM with Configuration.pushUpdatesToMemory();

Template Parameters
TData type of the configuration varable. Must match ConfigurationRegistry.h
