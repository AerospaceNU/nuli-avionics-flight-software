Detailed Description

template<typename T>
class ArgumentFlag< T >

Derived class of BaseFlag. Defines a flag that takes a fundamental type as an argument.

This class is derived from BaseFlag and implements a flag taking exactly one argument. Fundamental types are supported and explict specializations have been made for const char*.

Template Parameters
TA fundamental type, or const char*.
