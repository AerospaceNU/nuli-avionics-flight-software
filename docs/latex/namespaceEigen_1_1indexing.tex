\doxysubsection{Detailed Description}
The sole purpose of this namespace is to be able to import all functions and symbols that are expected to be used within operator() for indexing and slicing. If you already imported the whole \doxylink{namespaceEigen}{Eigen} namespace\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{using\ namespace\ }\mbox{\hyperlink{namespaceEigen}{Eigen}};\ }

\end{DoxyCode}
 then you are already all set. Otherwise, if you don\textquotesingle{}t want/cannot import the whole \doxylink{namespaceEigen}{Eigen} namespace, the following line\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{using\ namespace\ }\mbox{\hyperlink{namespaceEigen_1_1indexing}{Eigen::indexing}};\ }

\end{DoxyCode}
 is equivalent to\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{using\ }Eigen::all;}
\DoxyCodeLine{\textcolor{keyword}{using\ }Eigen::seq;}
\DoxyCodeLine{\textcolor{keyword}{using\ }\mbox{\hyperlink{namespaceEigen_a3a3c346d2a61d1e8e86e6fb4cf57fbda}{Eigen::seqN}};}
\DoxyCodeLine{\textcolor{keyword}{using\ }Eigen::lastN;\ \textcolor{comment}{//\ c++11\ only}}
\DoxyCodeLine{\textcolor{keyword}{using\ }Eigen::last;}
