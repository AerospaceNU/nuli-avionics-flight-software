class Eigen\+::\+Self\+Adjoint\+View$<$ \+\_\+\+Matrix\+Type, Up\+Lo $>$}
Expression of a selfadjoint matrix from a triangular part of a dense matrix. 


\begin{DoxyParams}{Parameters}
{\em Matrix\+Type} & the type of the dense matrix storing the coefficients \\
\hline
{\em Triangular\+Part} & can be either {\ttfamily \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cdaf581029282d421eee5aae14238c6f749}{Lower}} or {\ttfamily \doxylink{group__enums_gga39e3366ff5554d731e7dc8bb642f83cdafca2ccebb604f171656deb53e8c083c1}{Upper}} \\

This class is an expression of a sefladjoint matrix from a triangular part of a matrix with given dense storage of the coefficients. It is the return type of \doxylink{classEigen_1_1MatrixBase_aca50cf76ee44ab507c7fe3c5b1756ffc}{Matrix\+Base\+::selfadjoint\+View()} and most of the time this is the only way that it is used.
