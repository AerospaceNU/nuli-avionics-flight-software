\doxysubsection{Detailed Description}
Parses command line inputs into pre-\/defined Flag\+Groups and Flags. 

This class parses a user\textquotesingle{}s input, either from the command-\/line arguments or from input from stdin into Flag\+Groups containing sets of flags. A user should use the add\+Flag\+Group method to add sets of flags to a group. ~\newline
~\newline
A Flag\+Group is defined as a set of Flags. The first flag in a Flag\+Group is defined as a Flag\+Group\textquotesingle{}s \char`\"{}leader\char`\"{}. Among Flag\+Groups, only the leader flag must be unique, collision is permitted for members of a single Flag\+Group with another members of a different Flag\+Group.
