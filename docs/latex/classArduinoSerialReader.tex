\subsubsection*{template$<$unsigned N$>$\newline
class Arduino\+Serial\+Reader$<$ N $>$}
Reads newline-\/terminated text lines from the Arduino serial interface into a fixed-\/size buffer. 

This class provides a simple line reader implementation that reads characters from the global Arduino Serial interface. Characters are stored in an internal buffer up to a fixed capacity N. When a newline character (\textquotesingle{}~\newline
\textquotesingle{}) is received, the buffer is null-\/terminated and the function returns true. Optionally, received characters can be echoed back to the serial output as they are read. 
\begin{DoxyTemplParams}{Template Parameters}
{\em N} & The maximum size of the line buffer, including the null terminator. \\
