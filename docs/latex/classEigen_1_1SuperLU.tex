\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \+\_\+\+Matrix\+Type$>$\newline
class Eigen\+::\+Super\+LU$<$ \+\_\+\+Matrix\+Type $>$}
A sparse direct LU factorization and solver based on the \doxylink{classEigen_1_1SuperLU}{Super\+LU} library. 

This class allows to solve for A.\+X = B sparse linear problems via a direct LU factorization using the \doxylink{classEigen_1_1SuperLU}{Super\+LU} library. The sparse matrix A must be squared and invertible. The vectors or matrices X and B can be either dense or sparse.
