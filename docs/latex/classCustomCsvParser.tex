\doxysubsection{Detailed Description}
Parses a read-\/only CSV into a usable vector object. 

Provides functionality on retrieving data from the CSV. Enforces expectations of the CSV structure. The order of the input data must reflect the order in \mbox{\hyperlink{structCustomCsvParser_1_1FlightDataRow__s}{Flight\+Data\+Row\+\_\+s}}. This class presupposes the structure of the input CSV.
