\doxysubsection{Detailed Description}
An implementation of \doxylink{classBarometer}{Barometer} for the MS8607 barometer. 

Uses the Spark\+Fun PHT MS8607 Arduino library and implements an abstraction layer for our format. The datasheet can be found here\+: \href{https://cdn.sparkfun.com/assets/b/a/f/b/f/MS8607_Datasheet.pdf}{\texttt{https\+://cdn.\+sparkfun.\+com/assets/b/a/f/b/f/\+MS8607\+\_\+\+Datasheet.\+pdf}}
