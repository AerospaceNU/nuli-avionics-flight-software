\doxysubsection{Detailed Description}
Abstract base class for command-\/line flags. 

@\+TODO\+: Change up parse implementation. Current implementation by passing in argc, argv, and argv\+Pos are very hack-\/ey.

@\+TODO\+: If a flag is not set, don\textquotesingle{}t allow operations such as get\+Value. This is a little finicky without throwing an exception. I guess we can pass in the thing for where the returned value can go and then return an integer. OR, we can just have the user check if the flag\textquotesingle{}s value has been set each time before calling. Just depends on what type of "{}overhead"{} we want.

This class defines a common interface for different types of command-\/line flags, such as simple boolean flags and argument-\/based flags. Not intended for direct instantiation. \textbackslash{}derived
\begin{DoxyItemize}
\item {\ttfamily \doxylink{classSimpleFlag}{Simple\+Flag}}\+: Represents flags that are either set or unset (e.\+g., {\ttfamily -\/help}).~\newline

\item {\ttfamily \doxylink{classArgumentFlag}{Argument\+Flag}}\+: Represents flags that take an associated value (e.\+g., {\ttfamily -\/-\/threshold 5.\+0}). 
\end{DoxyItemize}
