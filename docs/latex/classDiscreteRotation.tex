\doxysubsection{Detailed Description}
Represents a 3D rotation composed only of 90° increments around the X, Y, or Z axes. 

This class allows for efficient manipulation of axis-\/aligned rotations by storing the mapping of input axes to output axes and their corresponding signs. It supports composing rotations, inverting them, and applying the transformation to 3D vectors.
