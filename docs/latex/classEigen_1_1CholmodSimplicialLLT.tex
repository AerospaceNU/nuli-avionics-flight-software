\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \+\_\+\+Matrix\+Type, int \+\_\+\+Up\+Lo = Lower$>$\newline
class Eigen\+::\+Cholmod\+Simplicial\+LLT$<$ \+\_\+\+Matrix\+Type, \+\_\+\+Up\+Lo $>$}
A simplicial direct Cholesky (\doxylink{classEigen_1_1LLT}{LLT}) factorization and solver based on Cholmod. 

This class allows to solve for A.\+X = B sparse linear problems via a simplicial LL\texorpdfstring{$^\wedge$}{\string^}T Cholesky factorization using the Cholmod library. This simplicial variant is equivalent to \doxylink{namespaceEigen}{Eigen}\textquotesingle{}s built-\/in \doxylink{classEigen_1_1SimplicialLLT}{Simplicial\+LLT} class. Therefore, it has little practical interest. The sparse matrix A must be selfadjoint and positive definite. The vectors or matrices X and B can be either dense or sparse.


\begin{DoxyTemplParams}{Template Parameters}
{\em \+\_\+\+Matrix\+Type} & the type of the sparse matrix A, it must be a \doxylink{classEigen_1_1SparseMatrix}{Sparse\+Matrix$<$$>$} \\
\hline
{\em \+\_\+\+Up\+Lo} & the triangular part that will be used for the computations. It can be Lower or Upper. Default is Lower.\\
\hline
\end{DoxyTemplParams}
\textbackslash{}implsparsesolverconcept

This class supports all kind of \doxylink{classEigen_1_1SparseMatrix}{Sparse\+Matrix$<$$>$}\+: row or column major; upper, lower, or both; compressed or non compressed.

\begin{DoxyWarning}{Warning}
Only double precision real and complex scalar types are supported by Cholmod.
